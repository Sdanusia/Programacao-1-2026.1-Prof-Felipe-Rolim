# Sistema "Troca Figurinhas" #

## 📋 Descrição do Projeto
Este projeto consiste em um sistema desenvolvido em Python para facilitar a troca de figurinhas do álbum da Copa do Mundo. O programa permite que os usuários cadastrem seus perfis, insiram a lista de figurinhas que possuem repetidas e criem uma lista de desejos com as fiurinhas que estão faltando. O sistema processa esses dados para cruzar informações e sugerir trocas compatíveis entre os usuários cadastrados.

## 🚀 Funcionalidades
- **Cadastro de Usuários:** Registro de colecionadores no sistema.
- **Gerenciamento de Inventário:** Cadastro de figurinhas repetidas.
- **Match de Trocas:** Algoritmo que cruza os dados dos usuários, aponta quem tem a figurinha que o outro precisa e envia mensagem com proposta.
- **Menu Interativo:** Interface via linha de comando (terminal) simples e intuitiva para navegação do usuário.

## 🛠️ Pré-requisitos
Para rodar este projeto, você precisa apenas do **Python 3.14** instalado em sua máquina, ou utilizar o VS Code, extraindo as pastas do arquivo .zip 

*Nota: Utilizamos apenas a biblioteca padrão do Python, não sendo necessária a instalação de nenhum pacote externo*

É necessário utilizar o menu para acessar as funcionalidades: cadastro de login e senha;
cadastro das figurinhas; informar a figurinha desejada;
